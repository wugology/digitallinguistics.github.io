Outline

    Text

    required The text is the fundamental unit of documentation. It represents a linguistic performance of a stretch of discourse in context. This may include a wide variety of linguistic genres: an elicitation session, a conversation, a narrative, a song, etc. 

    
  {
    "id" : "25-1",
    "title" : "Bear and Rabbit",
    "phrases" : [ { phrase 1 }, { phrase 2 }, ... ],
    "defaults" : {
      "orthography" : "SwadPhonemic",
      "speaker" : "Benjamin Paul"
    }
  }

    Text Identifier (ID)

    "id"

    Optional A unique string identifier for the text, (e.g. globally unique identifier, an integer, etc.)

    
  "id" : 25

    Title

    "title"

    Required A human-readable prose description of the content of the text.

    
  "title" : "Bear and Rabbit"

    Participants

    "participants"

    Optional A collection of all the individuals who contributed to this text in some way, for example through recording, digitization, analysis, transcription, or editing.

    
  "participants" : [ { participant 1 }, { participant 2 }, ... ]

    Properties

    "properties"

    Optional Specifies the default properties for the text. Every phrase within the text is then assumed to have the same properties unless specified otherwise. Properties that can be specified as defaults are:

    
  some code here

    Phrases

    "phrases"

    Required A collection of all the Phrases in the Text.

    
  "phrases": [ { phrase 1 }, { phrase 2 }, ... ]

    Phrase

    Required A segmentation of a Text. This standard imposes no requirements regarding how the linguist should implement this segmentation—linguists may choose to use Intonation Units, Turns, Sentences, or any other appropriate subdivision.

    
  {
    "transcriptions": {
      "SwadPhonemic" : "kunugu wey pu·p panš naʔank hi čuyiʔi",
      "ModPhonemic" : "kunugu wey puup panx naqank hi cuyiqi",
      "APAPhonemic" : "kunukʼu wey puːp panš naʔank hi čuyiʔi",
      "ModPractical" : "Kunugu wey puup panx naqank hi cuyiqi.",
      "SwadTranscript" : "kunuguˊ we pu·pˊ panš naʔankˊ hi ču·gšˊ […]"
    },
    
    "translations": {
      "SwadFree" : "The rabbit went to some people […]",
      "Literal" : "It is said the rabbit, going to the peopleʼs place […]"
    }
  }

    Transcriptions

    "transcriptions"

    Required A set of one or more Transcriptions for the Phrase.

    
  "transcriptions" : {
    "Swadesh-phonemic" : "kunugu wey pu·p panš naʔank hi čuyiʔi",
    "Modern-phonemic" : "kunugu wey puup panx naqank hi cuyiqi",
    "APA-phonemic" : "kunukʼu wey puːp panš naʔank hi čuyiʔi"
  }

    Transcription

    [name of orthography]

    Required A phonemic representation of the phrase in an orthography which is either in IPA or can be unambiguously mapped to IPA.

    
              "APA-phonemic" : "kunukʼu wey puːp panš naʔank hi čuːkʼš we kičk heːš ʔučpa ʔučaːšnakš"

    Speaker

    "speaker"

    Optional A unique identifier for the Person who spoke or otherwise created the phrase.

    
  "speaker": "JPH"
